project_info R Documentation

Get information about a project

Description

Get information about a project

Usage

project_info(zoltar_connection, project_url)

Arguments

zoltar_connection

A ZoltarConnection object as returned by new_connection()
project_url

URL of a project in zoltar_connection's projects

Value

A list of project information for the passed project_url

Examples

## Not run: 
  the_project_info <- project_info(conn, "https://www.zoltardata.com/api/project/9/")

## End(Not run)
