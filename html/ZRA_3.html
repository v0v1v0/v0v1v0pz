Dynamic Plots for Time Series Forecasting

Description

Dynamic Plots for Time Series Forecasting

Usage

ZRA(data, FP = 10, SL = c(0.8, 0.95), ...)

Arguments

data

(ts) Time series data.
FP

(numeric) Forecast period.
SL

(numeric) significance levels.
...

further arguments passed to the forecast-function.

Value

An Object of class "ZRA".

See Also

forecast, dygraphs

Examples

zra <- ZRA(fdeaths)
plot(zra, zero = TRUE)
