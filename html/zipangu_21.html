kana R Documentation

Create kana vector

Description

Generates a vector consisting of the elements of kana. Options exist for the inclusion of several elements.

Usage

kana(type, ...)

hiragana(
  core = TRUE,
  dakuon = FALSE,
  handakuon = FALSE,
  kogaki = FALSE,
  historical = FALSE
)

katakana(
  core = TRUE,
  dakuon = FALSE,
  handakuon = FALSE,
  kogaki = FALSE,
  historical = FALSE
)

Arguments

type

"hiragana" ("hira") or "katakana" ("kata")
...

Arguments passed on to hiragana

core

is include core kana characters.
dakuon

e.g. ga, gi, gu, ge, go
handakuon

e.g. pa, pi, pu, pe, po
kogaki

small character
historical

old style

Examples

kana(type = "hira", core = TRUE)
kana(type = "hira", core = TRUE, handakuon = TRUE)
